Code Projects

Here are some open source software projects I have done for fun.

Nebuloid

Nebuloid is a top down space shooter written in java. Fly around looking for colored nebulae to change your ship to shoot that color of laser, which is the only way to get through colored barriers. Destroy the barriers around the mother ship and blow it up to beat the level.

This could be a good way to learn about game loops and events to start on a game of your own.

Build and run the project by cloning the repo and running the included build.sh file.

PointAndPixel

PointAndPixel is a pixel art program written in java. You can either work free form or you can import an image to start from.

I wrote this after seeing a friend make pixel art in a spreadsheet.

Build and run the project by cloning the repo and running the included build_jar.sh file.

Picrypt

Picrypt is a steganography program written in java that embeds highly encrypted files into normal images. It uses public key encryption so keys can be easily shared between trusted parties. Then pictures with embedded data can be shared through normal services as long as they serve the raw image instead of a compressed or resized version.

Build and run the project by cloning the repo and running the included build.sh file.
